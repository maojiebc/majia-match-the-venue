# Observed Organizations

v0.1 跟踪的 19 家高端机构清单。完整结构化数据见 [../data/orgs.json](../data/orgs.json)。

## 选择标准

入选要求满足 **任一**：

1. 课程单价 ≥ 5 万元/人（含人均）
2. 企业内训单场预算 ≥ 30 万元
3. 历史选址中至少 5 次使用五星级酒店宴会厅
4. 学员/客户层级是 CEO / 总裁 / VP

## 抓取关键词字典

每家机构的 `keywords` 字段是用来做两件事的：

1. **搜索抓取**：在公众号/活动家/活动行搜索时拼这些词
2. **正文匹配**：抽取出来的活动文本里命中关键词 → 自动 link 到该 `org_id`

详细 keywords 见 `data/orgs.json` 每条 record 的 `keywords` 数组。

## 增删机构流程

1. PR 修改 `data/orgs.json`
2. 在本文件相应分类下加一行
3. 如果是新 category，先在本文件顶部"选择标准"里说明为什么纳入

## 排除清单（v0.1 不收）

| 机构类型 | 为什么不收 |
|---|---|
| 普通酒店宴会预订平台 | 不是培训机构 |
| 单一企业大学（华为大学/阿里湖畔学院旧址） | 闭门为主，公开证据稀少 |
| 单一城市本地培训机构 | 跨城市适用性差 |
| 婚宴/年会承办商 | 场景不匹配培训需求 |
